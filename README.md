## Getting Started

### Prerequisites

List of things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/aburi-ost/MAS_WE2_NoteApp.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
